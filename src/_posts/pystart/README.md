## 安装python

访问 https://www.python.org/downloads/release/python-391/ 最下面Windows installer (64-bit)安装

安装后cmd输入python 有结果则安装成功

python -> node
pip -> npm

## 安装pipenv

``` bash
$ pip install pipenv
```

速度太慢，使用清华镜像

``` bash
$ pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pipenv
```

安装后cmd输入pipenv 有结果则安装成功

pipenv -> yarn

## 安装项目依赖

进入项目 pipenv install

## 进入虚拟环境

pipenv shell
